## 1. 图形验证码（Image Captcha）

### 1.1 数字验证码
- **纯数字**: `1234`, `5678`, `0000`
- **4位数字**: `1234`
- **5位数字**: `12345`
- **6位数字**: `123456`
- **随机长度**: `1` 到 `10` 位数字

### 1.2 字母验证码
- **纯大写字母**: `ABCD`, `XYZ`
- **纯小写字母**: `abcd`, `xyz`
- **大小写混合**: `AbCd`, `XyZ`

### 1.3 数字+字母混合
- **数字+大写字母**: `A1B2`, `3C4D`
- **数字+小写字母**: `a1b2`, `3c4d`
- **数字+大小写混合**: `A1b2`, `3C4d`

### 1.4 中文验证码
- **纯中文**: `验证码`, `登录`
- **中文+数字**: `验证码123`, `登录456`
- **中文+字母**: `验证码ABC`, `登录xyz`

### 1.5 数学计算题验证码 ✅
- **加法**: `3+5=?` → `8`
- **减法**: `6-9=?` → `-3`
- **乘法**: `2*4=?` → `8`
- **除法**: `10/2=?` → `5`
- **中文运算符**: `8×3=？` → `24`, `15÷3=？` → `5`
- **带空格**: `6 - 9 = ?` → `-3`
- **中文问号**: `6-9=？` → `-3`
- **复杂表达式**: `(6+3)*2=?` (未来支持)

### 1.6 模糊/扭曲验证码
- **模糊文字**: 文字模糊不清
- **扭曲文字**: 文字被扭曲变形
- **干扰线**: 带有干扰线的验证码
- **噪点**: 带有噪点的验证码
- **背景干扰**: 复杂背景的验证码

### 1.7 彩色验证码
- **单色**: 单一颜色文字
- **多色**: 多种颜色文字
- **渐变**: 渐变颜色文字
- **背景色**: 彩色背景

### 1.8 动态验证码
- **GIF动画**: 动态显示的验证码
- **闪烁文字**: 文字闪烁效果
- **滚动文字**: 文字滚动效果

### 1.9 特殊字符验证码
- **符号**: `!@#$%^&*()`
- **标点**: `.,;:!?`
- **特殊字符**: `@#$%`

### 1.10 验证码变体
- **刷新按钮**: 点击刷新获取新验证码
- **语音播放**: 点击播放语音验证码
- **提示文字**: 带有提示文字的验证码

## 📊 系统支持情况总结

### ✅ 完全支持（自动处理）
1. **图形验证码（含数学计算题）** - 视觉模型识别 + 数学计算
2. **滑动验证码** - 视觉模型识别缺口 + 分段滑动
3. **点选验证码** - 视觉模型识别点击目标
4. **拼图验证码** - 视觉模型识别缺口位置
5. **旋转验证码** - 视觉模型识别旋转角度

### ⚠️ 部分支持（需要配置）
1. **图形验证码（OCR）** - 需要配置OCR服务
2. **短信验证码** - 需要集成接码平台
3. **邮箱验证码** - 需要集成邮箱服务
4. **语音验证码** - 需要集成语音识别服务

### ❌ 需要第三方服务
1. **reCAPTCHA** - 需要2captcha等第三方服务
2. **hCaptcha** - 需要2captcha等第三方服务
3. **Cloudflare Turnstile** - 需要2captcha等第三方服务

### 🔄 需要人工介入
1. **复杂图形验证码** - 模糊、扭曲、干扰严重的验证码
2. **行为验证码** - 需要分析用户行为
3. **混合验证码** - 需要完成多个验证步骤

## 📝 注意事项

1. **法律合规**: 自动化处理验证码可能违反某些网站的服务条款，请确保合法使用
2. **识别准确率**: 不同验证码类型的识别准确率不同，需要根据实际情况调整
3. **性能考虑**: 视觉模型识别需要时间，需要考虑性能影响
4. **成本考虑**: 使用第三方服务（如2captcha）会产生费用
5. **扩展性**: 系统设计支持扩展新的验证码类型和处理方式
