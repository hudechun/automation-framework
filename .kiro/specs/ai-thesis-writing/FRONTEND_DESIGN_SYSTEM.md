# AI论文写作系统 - 前端设计系统

## 🎨 色彩系统

### 主色调 (学术蓝)
```css

--primary-light: #40a9ff; /* 浅色 */
--primary-dark: #096dd9;  /* 深色 */
```

### 辅助色
```css
--success: #52c41a;  /* 成功 - 绿色 */
--warning: #faad14;  /* 警告 - 橙色 */
--danger: #f5222d;   /* 危险 - 红色 */
--info: #13c2c2;     /* 信息 - 青色 */
```

### 中性色
```css
--text-primary: #262626;    /* 主文本 */
--text-secondary: #595959;  /* 次要文本 */
--text-disabled: #bfbfbf;   /* 禁用文本 */
--border: #d9d9d9;          /* 边框 */
--background: #f0f2f5;      /* 背景 */
```

## 📱 页面结构

### 1. 会员管理页面
**布局**: 卡片式套餐展示 + 表格管理  
**特点**: 
- 套餐卡片突出价格和特性
- 使用徽章标识推荐套餐
- 配额使用进度条可视化

### 2. 论文管理页面
**布局**: 列表 + 详情侧边栏  
**特点**:
- 论文状态流程可视化
- 快速操作按钮
- 实时生成进度

### 3. 模板管理页面
**布局**: 网格卡片 + 预览  
**特点**:
- 模板缩略图预览
- 标签分类
- 一键应用

### 4. 订单管理页面
**布局**: 时间线 + 表格  
**特点**:
- 订单状态时间线
- 支付状态标识
- 快速退款操作

### 5. 支付管理页面
**布局**: 配置表单 + 交易记录  
**特点**:
- 支付渠道图标
- 配置状态指示

## 🎭 组件规范

### 按钮
- **主要按钮**: 蓝色，用于主要操作
- **次要按钮**: 白色边框，用于次要操作
- **危险按钮**: 红色，用于删除等危险操作
- **文本按钮**: 无边框，用于辅助操作

### 表单
- **标签宽度**: 100px
- **输入框高度**: 32px
- **必填标识**: 红色星号
- **错误提示**: 红色文字，表单下方

### 表格
- **行高**: 48px
- **斑马纹**: 奇数行白色，偶数行浅灰
- **悬停**: 浅蓝色背景
- **操作列**: 固定在右侧

### 卡片
- **阴影**: 0 2px 8px rgba(0,0,0,0.1)
- **内边距**: 24px
- **圆角**: 8px
- **悬停**: 阴影加深

## ♿ 无障碍设计

### 对比度
- 文本对比度 ≥ 4.5:1
- 大文本对比度 ≥ 3:1

### 键盘导航
- 所有交互元素可通过Tab键访问
- 焦点状态清晰可见

### 屏幕阅读器
- 使用语义化HTML
- 添加aria-label属性
