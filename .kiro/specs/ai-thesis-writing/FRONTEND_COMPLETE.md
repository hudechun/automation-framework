## 📊 技术实现

### 使用的技术栈
- **框架**: Vue 3 + Composition API
- **UI库**: Element Plus
- **状态管理**: Pinia (RuoYi内置)
- **路由**: Vue Router
- **HTTP**: Axios
- **样式**: SCSS

### 核心组件
- **Table**: 数据表格
- **Form**: 表单
- **Dialog**: 对话框
- **Card**: 卡片
- **Steps**: 步骤条
- **Progress**: 进度条
- **Tag**: 标签
- **Upload**: 上传
- **Icon**: 图标

### 设计模式
- **响应式数据**: ref/reactive
- **异步操作**: async/await
- **权限控制**: v-hasPermi指令
- **字典数据**: useDict组合式函数
- **分页**: pagination组件
- **搜索**: right-toolbar组件

## 📱 响应式设计

### 断点设置
```scss
xs: <768px   // 移动端
sm: ≥768px   // 平板
md: ≥992px   // 小屏桌面
lg: ≥1200px  // 大屏桌面
```

### 适配策略
- **移动端**: 单列布局，隐藏次要信息
- **平板**: 双列布局，保留主要功能
- **桌面**: 多列布局，完整功能
